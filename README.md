# Graph Neural Time Series Forecasting (GCN + GRU)

A clean, minimal, fully working implementation inspired by  
**Graph Neural Time Series (2023)**.

This project builds a synthetic graph, generates multivariate time series,  
and trains a **GCN + GRU** hybrid model to forecast future values.
